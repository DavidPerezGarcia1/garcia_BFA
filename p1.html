This art piece I made using Java by using a paint function to make my own custom brush.
I made three of them: glitchy, smooth and random rotate. The reason I wanted three ways
different from each other is because I wanted to combine/mix brushes that are normally not compatible,
to make a good looking art image. For glitchy brush, it’s made from broken input of a triangle, which
rather forms into more of a rectangle, this allows me to have a more broken/glitchy brush look. For
smooth brush, I use more weight of the strokes and line function, for random rotate brush, use rotate
and random function to get a more uneditable brush.
